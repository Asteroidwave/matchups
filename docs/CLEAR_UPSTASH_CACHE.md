# How to Clear Upstash/Redis Cache ## Option 1: Automatic (Recommended) The updated `clear-cache.ts` script now **automatically tries to clear Redis/Upstash cache** when you run it: ```bash npx tsx scripts/clear-cache.ts ``` **Note**: This requires: - Your backend server to be running on `http://localhost:3001` - You to be logged in as an admin (the script will try to get your session) If the automatic clear fails, you'll see a warning but the Supabase data will still be cleared. ## Option 2: Via Backend API If your backend is running, you can clear Redis cache via the API: ```bash # First, get your admin token (you need to be logged in as admin) # Then call: curl -X POST http://localhost:3001/api/admin/contests/cache/clear \ -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \ -H "Content-Type: application/json" ``` ## Option 3: Manual via Upstash Dashboard 1. Go to [Upstash Console](https://console.upstash.com/) 2. Select your Redis database 3. Go to the "Data Browser" or "CLI" tab 4. Use the Redis CLI to delete cache keys: ```redis # Delete the contests cache key DEL contests:active # Delete all entries cache keys (if you want to clear all track data) DEL entries:* # Delete all results cache keys DEL results:* # Or delete all keys (be careful!) FLUSHALL ``` ## Option 4: Create a Simple Script You can also create a small script to clear Redis directly: ```typescript // scripts/clear-redis.ts import { Redis } from '@upstash/redis'; import dotenv from 'dotenv'; import path from 'path'; dotenv.config({ path: path.resolve(__dirname, '../backend/.env') }); const redis = new Redis({ url: process.env.UPSTASH_REDIS_REST_URL!, token: process.env.UPSTASH_REDIS_REST_TOKEN!, }); async function clearRedis() { console.log(' Clearing Redis cache...'); // Delete specific keys await redis.del('contests:active'); console.log(' Cleared contests:active'); // Or delete all keys matching a pattern // Note: Upstash Redis doesn't support KEYS command directly // You'd need to track keys manually or use SCAN console.log(' Redis cache cleared!'); } clearRedis(); ``` ## What Gets Cached in Redis? The main cache keys are: - `contests:active` - Active contests list - `entries:TRACK:DATE` - Track entries (e.g., `entries:GP:2025-11-02`) - `results:TRACK:DATE` - Track results (e.g., `results:GP:2025-11-02`) ## When to Clear Redis Cache? Clear Redis cache when: - You've cleared Supabase data and want fresh data - You've made changes to contest visibility - You want to force fresh data from MongoDB - Testing the full flow from scratch ## Quick Reference **Fastest way**: Just run the updated script: ```bash npx tsx scripts/clear-cache.ts ``` It will: 1. Clear Supabase data 2. Try to clear Redis cache automatically 3. Tell you if it failed and how to clear manually
